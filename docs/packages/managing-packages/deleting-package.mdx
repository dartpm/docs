When the package is replaced by new package or just removed, we recommend deleting unused packages.

1. On the dartpm website, go to the package page https://dartpm.com/package/package-name

2. On the package page, click Settings.
<img src={packageSettings} style={{ border: `1px solid black` }}/>

3. Click the delete package button.
<img src={packageSettingsMakePublic} style={{ border: `1px solid black` }} width="500"/>
